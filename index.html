Integración Google Sheets

La integración con Sheets requiere desplegar un Google Apps Script como Web App. Ver script a copiar ▾ 


      
    

    

      Pasos de despliegue
      
    
        
  1. Abrí un Google Sheet nuevo
    2. 
        
  2. Menú: Extensiones → Apps Script
    3. 
        
  3. Borrá el código por defecto y pegá el script (link arriba)
    4. 
        
  4. Cambiá cambia-este-token por tu token (mismo que pegás acá)
    5. 
        
  5. Guardá con el disquete, nombrá el proyecto
    6. 
        
  6. Desplegar → Nueva implementación → Aplicación web
    7. 
        
  7. Ejecutar como: Yo
    8. 
        
  8. Quién tiene acceso: Cualquier usuario (crítico)
    9. 
        
  9. Aceptá permisos, copiá la URL que termina en /exec
    10. 
        
  10. Pegá la URL acá abajo y tocá Probar conexión
    11. 
      

      

        ⚠ Si modificás el script después, usá Administrar implementaciones → lápiz → Nueva versión (no nueva implementación, eso cambia la URL).
